1. 在ESMM的基础上，加了一个counterfactual cvr塔的训练，该塔只有正样本没有负样本
2. 在ESMMloss的基础上，新加一个DCMTloss，这个loss做了两部分改进，第一部分是分别对factual loss和counterfactual loss做IPW处理，即分别除以是否点击的预测概率，第二部分加了一个正则项（作者称为软约束）
3. 其他就是IPW的常规操作self-norm等
